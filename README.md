# 📊 Impact100 Visual Data Exploration Environment

An interactive data visualisation project built for **Impact100 Sydney** as part of BUSA3021 Business Analytics Project at Macquarie University. The application enables stakeholders to visually explore volunteering opportunities, funded community projects, and global Impact100 chapters.

---

## 🌏 Project Overview

This project delivers a multi-stakeholder **visual analytics (VA) application** designed using human-centred design thinking. Data was ethically harvested from two public sources — Impact100 Global and the Australian Communities Foundation (ACF) funding platform — and transformed into interactive Tableau dashboards tailored to three distinct user personas.

---

## 👥 Design Personas

The dashboards were designed around the decision-making needs of three user archetypes:

| Persona | Role | Focus |
|---|---|---|
| Farida | BA Graduate & Impact100 Enthusiast | Exploring global Impact100 chapters and projects |
| Lee | CSR Manager | Discovering organisations seeking funding via ACF |
| Anisha | Final-Year MQ University Student | Identifying volunteering & engagement opportunities for MQ students |

Each persona drives a dedicated set of non-trivial data exploration questions answered visually within the application.

---

## ✨ Features

- **Interactive Tableau dashboards** — one per persona, each answering 3 non-trivial analytical questions
- **Ethically harvested dataset** — 50+ organisations sourced from Impact100 Global and ACF, stored in Excel
- **Design thinking methodology** — persona-based UX design applied throughout
- **Data quality analysis** — syntactic, semantic, and pragmatic issues identified and addressed
- **Ethics review** — PAPA framework applied for data ethics; visual ethics considerations documented
- **"Do No Harm" principles** — equity and diversity guidelines applied to all visualisations
